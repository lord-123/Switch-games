# How to play breakout

| Action  | PC | Switch |
|---------|----|--------|
| Pause  | Space/Enter  | Any button not in use |
| Move  | Mouse/Arrow keys  | Left stick/DPAD |
| Fullscreen | Up arrow | Up arrow |
| Toggle music | Down arrow | Down arrow |
