# How to play pong

| Action  | PC | Switch |
|---------|----|--------|
| Pause  | Space/Enter  | Any not in use |
| Move  | Mouse/Arrow keys  | Left stick/DPAD |
| Fullscreen | Right arrow | Right arrow |
| Toggle audio | Left arrow | Left arrow |
