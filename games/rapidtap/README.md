# How to play rapid tap

Simply tap the squares that light up as fast as possible!

## settings

| Action | PC | Switch |
|--------|----|--------|
| Tap | Click | Tap |
| Fullscreen | Up arrow | Up arrow |
| Mute | Down arrow | Audio not currently working |
